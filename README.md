# 📄 AI PDF Summarizer

A web app that uses AI to instantly summarize any PDF document.
Built with Python, Streamlit, and Groq AI.

## 🌐 Live Demo
[Click here to try the app](https://ai-pdf-summarizer-p7mumauft5hrgw7ny86srq.streamlit.app)

## ✨ Features
- 📄 Upload any PDF document
- 🎯 Choose summary style — Bullet Points, Short or Detailed
- 🌐 Get summary in English, Malayalam, Hindi or Tamil
- 🙋 Ask questions about the PDF
- ⬇️ Download summary as a formatted text file
- 📊 See PDF statistics — pages, words processed
- 🛡️ Handles errors gracefully

## 🛠️ Built With
- Python
- Streamlit
- Groq AI (LLaMA 3.3)
- PDFPlumber

## 🚀 How to Run
1. Clone this repository
2. Install requirements:
   pip install streamlit groq pdfplumber
3. Add your Groq API key in the code
4. Run the app:
   streamlit run day8.py
